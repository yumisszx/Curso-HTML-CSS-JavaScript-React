# 🚀 Curso HTML, CSS, JavaScript e React

Este repositório contém os arquivos dos projetos desenvolvidos durante o curso **Formação Front-end - HTML, CSS, JavaScript, React e +** do Matheus Battisti na Udemy.

## 📚 Sobre o projeto

O objetivo dos projetos é colocar em prática tudo que foi aprendido em cada etapa do curso, e neste repositório estão todos os projetos desenvolvidos.

## 🛠️ Tecnologias utilizadas

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6+)
- ⚛️ React
- 🧰 Git & GitHub


## 📌 Conteúdos abordados

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/yumisszx/Curso-HTML-CSS-JavaScript-React.git

cd Curso-HTML-CSS-JavaScript-React

npm install
npm start
```
